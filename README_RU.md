# DevLog — Трекер рабочего времени

Одностраничное приложение для учёта рабочего времени и управления задачами по проектам.  
Создано на Angular 19 как pet-проект — без backend, без базы данных, работает полностью в браузере.

---

## Быстрый старт

**Требования:** Node.js 18+, npm

```bash
# Клонировать репозиторий
git clone https://github.com/DmitriiKasapov/pet-angular.git
cd pet-angular

# Установить зависимости
npm install

# Запустить сервер разработки
npm start
```

Открыть адрес, указанный в терминале, в браузере.

> При первом запуске приложение автоматически загружает демо-данные: 3 проекта, 11 задач, 15 записей worklog и 5 комментариев — можно сразу изучить весь функционал.

---

## Хранение данных

Все данные хранятся в **localStorage** — ничего не отправляется на сервер.  
Чтобы сбросить данные к исходным демо-данным: открыть DevTools → Application → Local Storage → удалить все ключи, начинающиеся с `worklog-`.

## Функциональность

### Недельная доска `/board` ← страница по умолчанию
- Визуальная временна́я сетка текущей недели (Пн–Вс)
- Все записи worklog отображаются как блоки на сетке
- **Клик по колонке** — создать запись для этого дня
- **Перетащить блок** между днями — перенести запись
- **Растянуть блок** за нижний край — изменить продолжительность
- **Редактировать или удалить** запись через кнопки при наведении
- Навигация по неделям: Назад / Вперёд; кнопка «Today» возвращает к текущей

### Проекты `/projects`
- Список всех проектов с названием, кодом и описанием
- **Создать проект** — название, короткий код (например `INT`), описание, цвет
- **Поиск** проектов по названию, коду или названию задачи
- Клик по проекту открывает его страницу

### Страница проекта `/projects/:id`
- **Вкладка «Задачи»** — список задач со статусами и залогированными часами
  - Поиск задач по коду или названию
  - **Создать задачу** — название, описание, статус, оценка в часах
  - Клик по задаче открывает её страницу
- **Вкладка «Аналитика»** — часы по задачам с прогресс-барами, сортировка по убыванию

### Страница задачи `/tasks/:id`
- Полная информация о задаче: код, название, описание, статус, оценка
- **Сменить статус** — Планируется → В работе → Выполнено
- **Залогировать время** — дата, время начала, продолжительность, комментарий
- Список записей с суммарными часами
- **Комментарии** — добавить, редактировать, удалить

### Аналитика `/analytics`
- Итог за неделю: общие часы и количество записей
- **По дням** — столбчатый график для каждого дня недели
- **По проектам** — цветные полоски, сортировка по убыванию часов
- **По задачам** — таблица с кодом, названием, проектом, статусом и часами
- Навигация по неделям; кнопка «Today» возвращает к текущей
